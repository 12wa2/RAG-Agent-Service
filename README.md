# 基于 RAG 与 Agent 的垂直领域智能客服系统
> 本项目是一个面向垂直领域（如扫地机故障排查）的工业级智能客服框架。采用前后端彻底分离的微服务架构，集成高阶 RAG 检索链路与并发状态控制机制。

## ✨ 核心特性 / Features

* **前后端分离微服务**：使用 FastAPI 独立封装核心推理与检索逻辑，对外提供标准 RESTful API，Streamlit 仅作纯粹的 UI 渲染。
* **极速流式响应 (SSE)**：基于 Server-Sent Events 协议的流式接口，前端实时解析 chunk 数据流，实现极低首字响应时间 (TTFT) 的“打字机”交互。
* **高阶检索架构 (RAG)**：采用 `多路召回 + 二次重排` 机制。结合 BM25 与 ChromaDB 向量库粗排，引入 **BGE-Reranker** 进行语义精排，配合 ReAct 压制幻觉。
* **高并发防呆控制**：在长耗时流式推理场景下，于数据库写入层引入“现查、现改、现存”的临界区刷新机制，解决多端数据版本覆盖问题。
* **长文本记忆管理**：实现“滑动窗口 + LLM 动态摘要”的长文本压缩算法，突破大模型 Token 限制。
